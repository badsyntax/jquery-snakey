jQuery Snakey

  • Overview

Instructions

Use the keyboard arrow keys to control the snake. Eat as many cherries as you can to achieve a top score for the map. The speed and length of the snake will increase with every cherry eaten, the level ends when you slither into yourself. Have fun!

Shortcut Keys

  • P : Pause/Resume Game
  • N : New Game
  • G : Toggle Grid
